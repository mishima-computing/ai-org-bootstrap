# Antigravity Provisional Area

Antigravity support is provisional.

This area is reserved for future Antigravity role adapters and skills after the local CLI contract is verified.

Initial allowed use cases:

- visual review
- browser evidence
- UI exploration
- screenshot-based review
- recording-based review

Initial prohibited use cases:

- authentication
- authorization
- database work
- migrations
- production operations
- deployment
- secrets
- final audit authority
- adoption authority

Do not add Antigravity agents or skills until a smoke test confirms the local CLI behavior and output contract.
